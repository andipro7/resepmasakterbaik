Gongso Tauge Tomat

Ceritanya ini saya pas lagi libur 3 hari di rumah. Iseng2 nemu kacang hijau di kulkas, gatel deh langsung ingin buat tauge. Udah buat tauge, ternyata selama 3 hari itu tauge blm panjang dan saya pun harus kembali ke kos melanjutkan studi. Alhasil ibu saya yang memanen tauge dan disimpan dalam kulkas. Beberapa pekan kemudian saya pulang. Karena pengen masak, saya lihat isi kulkas. Ibu saya bilang "itu tauge km kmrn tak taruh di kulkas". Kemudian saya ambil tauge, ternyata saya lihat taugenya panjang dan kurus (jadi seperti mie), panjang akibat telat masa pengambilannya. Akhirnya saya memasaknya menjadi gongso bersama tomat. #resepAnaQnh #gongso #gongsotauge #homemade
