Sayur Kecambah Ala Rumahan Simpel

Ibu suka sekali masak sayur kecambah. Dulu saya tidak suka sayur kecambah, soalnya ibu kalalu bikin 1 panci kecambah semua dan saya tidak suka lihatnya. Awalnya saya disuruh masak kecambah banyak sekali, saya tidak suka lihatnya tapi makanan kalau tidak dicicipi rasanya kurang. Akhirnya dengan terpaksa saya cicipi dan ternyata rasanya enak. Lama2 saya suka sayur kecambah entah kenapa. Ternyata rasanya kriuk-kriuk enak, cara membuatnyapun simpel sekali dan saya suka. #SemuaTentangIbu #SayurKesukaanIbu #DariPengenJadiBisa #SatuResepSatuPohon #OneRecipeOneTree #TeamTrees
